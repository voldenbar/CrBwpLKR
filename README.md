# 项目简介

## 核心功能
- 基于 YOLOv8 的 3D 打印缺陷目标检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 自定义置信度和 IOU 阈值
- 可选择显示标签和原图

## 适用场景
- 3D 打印质量检测
- 工业缺陷识别
- 制品质量监控
- 生产过程质量控制

## 技术特点
- 采用 Python 3.9 开发
- 依赖环境包括 opencv-python、PyQt5、ultralytics、torch1.9 等
- 包含 11028 张标注数据集，覆盖四类缺陷：Cracks、Layer-shift、Spaghetti、Stringging
- 提供完整源码、训练好的模型、数据集及运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 配备图形用户界面，操作简便
